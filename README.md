# Cornix Typing

[Cornix](https://en.zfrontier.com/products/in-stock-cornix-tented-low-profile-split-ergo-keyboard-by-jezail-funder)（Jezail Funder製 分割キーボード）専用のタイピング練習サイト。

あなた自身のキーマップを読み取り、次に押すべきキー（レイヤーキー・Shiftの押し順を含む）をキーボード図上にインジケータ表示します。

## 特徴

- **キーマップ連動**: WebHID（Vialプロトコル）でUSB接続中のCornixから直接読み取り、または `.vil` ファイルのドラッグ＆ドロップ
- **3+1モード**: 英単語・英文 / 日本語ローマ字（ヘボン式・訓令式両対応）/ 記号・レイヤー練習 / ミックス
- **寿司打スタイル**: 30秒・60秒選択、30コンボごとに+1秒ボーナス、総合スコアとランク判定
- **効果音**: タイプ音・コンボ音・ミス音（ON/OFF切替可）

## 使い方

ブラウザ（Chrome / Edge 推奨。WebHIDが必要）で `index.html` を開くだけ。ビルド不要の単一HTMLファイルです。

公開ページ: `https://<ユーザー名>.github.io/<リポジトリ名>/`
